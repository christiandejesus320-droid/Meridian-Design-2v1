POLÍTICA DE
PRIVACIDAD.

Explicamos qué información recibe el sitio público, para qué se usa y cómo pedir una revisión o eliminación.

🔒
DATACARERLSAI
01 / DATOS QUE RECIBIMOS

Contexto para ayudarte.

El chat público puede guardar el texto de la conversación, un correo, una URL y metadatos técnicos básicos para mantener el contexto y responder mejor. El formulario de contacto guarda nombre, correo, plan, enlace y mensaje para que el equipo pueda revisar la solicitud.

02 / USO

Solo para operar Meridian.

Usamos esa información para responder, analizar el enlace enviado de forma segura, revisar solicitudes y mejorar el servicio. No pedimos contraseñas, claves API ni datos de pago en el chat público.

03 / CONTROL

Tu contexto no es un permiso global.

Las tablas públicas de memoria están protegidas con RLS y el navegador nunca recibe la clave secreta de Supabase o Gemini. Para pedir acceso, corrección o eliminación, usa Contacto.
